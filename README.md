This repository contains the solution for Task 3 of the Skill Craft Technology Software Development Internship:
“Create a program that solves Sudoku puzzles automatically.”
The project implements a Python-based Sudoku Solver using the Backtracking Algorithm, capable of solving any valid 9×9 Sudoku grid.
Task Objective
-Build a program that takes an unsolved 9×9 Sudoku grid as input
-Use an algorithm to fill the missing cells
-Display the completed solution
